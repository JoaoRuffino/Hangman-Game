# Hangman-Game
A Hangman Game for you and your friends.
Project made using
- 📙 JavaScript
- 📘 HTML5
- 📗 CSS

## Tips for playing:
- ⚠️ In custom mode, enter the word in capital letters;
- ⚠️ If you want to know the word, just access the browser console
- 📢 The game is in Portuguese
